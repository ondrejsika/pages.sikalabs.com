# MDX Example Page

Welcome to the **comprehensive example** showcasing the power of MDX! This page demonstrates how you can combine Markdown with React components.

<InfoBox title="What is MDX?" type="info">
MDX allows you to use JSX and React components directly in your markdown content. Write content naturally while adding interactive elements where needed.
</InfoBox>

## Markdown Basics

MDX supports all standard Markdown syntax. Here's what you can do:

### Text Formatting

You can use **bold text**, *italic text*, ~~strikethrough~~, and `inline code`.

### Lists

**Unordered lists:**
- Simple and clean syntax
- Easy to read and write
- Perfect for documentation
  - Nested items work too
  - Just indent with spaces

**Ordered lists:**
1. First item
2. Second item
3. Third item

### Code Blocks

```javascript
// JavaScript code with syntax highlighting
function greet(name) {
  console.log(`Hello, ${name}!`);
  return true;
}

greet('World');
```

```bash
# Shell commands

npm run dev
```

## Getting Started

Want to create a page like this? Just add a `.mdx` file to the `content/` folder:

```bash
# Create a new page
echo "# My Page" > content/mypage.mdx
```

Then visit `/mypage` and your content is live!

<InfoBox title="Pro Tip" type="success">
You can use React components without importing them. They're automatically available in all your MDX files!
</InfoBox>
