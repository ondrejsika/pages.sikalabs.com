# sikalabs/slu

## What is slu?

slu is a open-source tool from SikaLabs which simplify lots of common tasks for developers and DevOps engineers.

slu is open-source under the MIT License and available on Github on https://github.com/sikalabs/slu.

## Install slu

### Install slu on Mac

```bash
brew install sikalabs/tap/slu
```

### Install slu on Linux

```bash
curl -fsSL https://raw.githubusercontent.com/sikalabs/slu/master/install.sh | sudo sh
```
